# viron-demo

# QuickStart

## 前準備

- [Docker for Mac](https://docs.docker.com/docker-for-mac/) をインストール


## 起動 on Docker

```
$ cp -ip .env.template .env
$ # 必要に応じて.envを編集してください
$ npm run dressup
```

### ブラウザアクセス

- [api#swagger.json](https://localhost:3000/swagger.json)

# Tools

$ npm run
